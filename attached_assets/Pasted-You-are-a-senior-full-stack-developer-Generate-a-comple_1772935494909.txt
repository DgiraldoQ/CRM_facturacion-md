Invoices

- quantity
- price

Calculate automatically:
- subtotal
- iva
- total

When saving an invoice:

1. Generate the invoice number using the business resolution:
   prefix + next sequential number

2. Generate a simulated CUFE using SHA256 hash of:
   invoice number + date + total + customer NIT

3. Generate a simplified UBL-like XML structure containing:
   - company data
   - customer data
   - products
   - totals

4. Simulate sending to DIAN (mock):
   Instead of a real provider, just generate the CUFE.

5. Generate a QR code with the CUFE.

6. Generate a PDF invoice including:
   - company data
   - customer data
   - product table
   - totals
   - CUFE
   - QR code

Use jsPDF or @react-pdf/renderer.

7. Save invoice in database with:

- number
- cufe
- xml (string)
- pdf (base64 or file path)
- date
- subtotal
- iva
- total
- businessId
- customerId
